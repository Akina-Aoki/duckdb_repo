# Data Modeling with Python and Docker
**Used the SQL repo for practice**

- Create a new folder `simple_container`
- Create the follwing files inside the folder with their respective content:
  - `app.py`
  - `requirements.txt`
  - `Dockerfile`

- In the terminal run `docker build -t first-python-app .`
- `first-python-app`is the name of the image created. Create own name.

- Check Docker desktop to see that the `first-python-app`   image is created.

- Run `docker image` to see the list of images created.

- `docker run first-python-app` to run the container.
